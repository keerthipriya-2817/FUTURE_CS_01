# Task 1 – Vulnerability Assessment Report

## Overview
This project was completed as part of the Future Interns Cyber Security Internship 2026.

The objective was to perform an ethical, read-only vulnerability assessment on a publicly accessible website and prepare a professional security assessment report.

---

## Target Website

- Website: http://scanme.nmap.org
- Scope: Public pages only
- Type of Assessment: Passive Vulnerability Assessment

---

## Tools Used

- Nmap
- OWASP ZAP (Passive Scan)
- Browser
- Git & GitHub

---

## Methodology

1. Identified open ports using Nmap.
2. Detected running services and versions.
3. Performed a passive web application scan using OWASP ZAP.
4. Collected security findings.
5. Classified the risks.
6. Suggested remediation steps.

---

## Key Findings

- Open network services identified
- Missing security headers
- Server version disclosure
- Information disclosure
- Directory browsing detected
- Other low-risk configuration issues

---

## Files Included

- ZAP_Report.html
- Nmap_Output.txt

---

## Disclaimer

This assessment was performed only against an authorized public test website using passive techniques. No exploitation, brute-force attacks, or destructive testing were performed.
